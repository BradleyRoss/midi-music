Resources

The purpose of this page is to collect resources for information on MIDI programming using Java.

Web Sites

If you enter the words "music tech" in the Google search engine, you will find a large number of web sites listed. However, it appears that a number of these have not been updated for some time.

  • Audio Melody
  • Java Sound Resources: Examples: MIDI: Playback and Recording has a number of examples. However, it appears that Java Sound Resources ( http://www.jsresources.org ) has not been updated since 2013. The SourceForge repository can be found at http://jsresources.cvs.sourceforge.netjsresources/.
  • Music Radar - This site contains a separate page for technology.
  • Music Software Development Blog
  • Music Software Development
  • MusicTech.net
  • MusicTech magazine
  • sonicstate.com

  • Overtone Google Group
