Displaying MIDI Messages in Log Window

This menu item displays Turn Log Window off when the log option is selected and Turn Log Window on at other times.

Playing a MIDI File on Synthesizer

  • Use File -> Choose Input File to specify the MIDI file to be used. Of the MIDI devices displayed by the application, select the software sequencer in the Source column and the synthesizer in the Destination column.
  • The menu item Connection -> Open Connection is then used to play the music.
  • The menu item Connection -> Close Connection can then be used to close the task so that other activities can be carried out.

  • Use File -> Choose Output File to specify the MIDI file to be written. Of the MIDI devices displayed by the application, select the keyboard in the Source column and the sequencer in the Destination column.
